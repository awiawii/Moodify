# Moodify Landing Page (React)

Welcome to the Moodify Landing Page repository built with React.js! This project showcases a responsive landing page for Moodify, a fictional mental well-being companion application.

![Photo-1](https://github.com/awiawii/Moodify/blob/Cloud-Computing-Landing-Page/moodify.png)

## Overview

Moodify Landing Page is designed to promote our product Moodify. This React-based landing page offers an interactive experience to learn about Moodify's features, benefits, and the teams behind it.

## Technologies Used

- React.js
- React Router
- HTML5
- CSS3
